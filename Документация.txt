# Документация к скрипту оптимизации Windows Zerio Command

## Общее описание
Этот скрипт предназначен для комплексной оптимизации операционной системы Windows. Он выполняет множество настроек, которые ускоряют работу системы, отключают ненужные функции, улучшают безопасность и конфиденциальность.

## Основные функции скрипта

### 1. Оптимизация производительности
- Настраивает параметры реестра для улучшения работы памяти и процессора
- Оптимизирует работу файловой системы
- Настраивает параметры TCP/IP для лучшей сетевой производительности
- Включает схему максимальной производительности электропитания
- Оптимизирует работу сетевых адаптеров
- Настраивает параметры ядра и управления памятью

### 2. Отключение телеметрии и слежки
- Отключает службы телеметрии Windows
- Запрещает сбор диагностических данных
- Отключает рекламные функции Windows
- Блокирует отправку отзывов и предложений
- Отключает телеметрию NVIDIA (если установлена)

### 3. Очистка системы
- Удаляет временные файлы
- Очищает кэш DNS
- Удаляет старые точки восстановления
- Очищает журналы событий
- Удаляет старые обновления Windows

### 4. Отключение ненужных служб
- Отключает OneDrive
- Отключает службы Xbox
- Отключает фоновые задачи
- Отключает ненужные сетевые службы
- Отключает планировщик задач Microsoft

### 5. Улучшение безопасности
- Отключает устаревший протокол SMBv1
- Отключает удаленный рабочий стол
- Отключает IPv6 (по желанию)

### 6. Дополнительные настройки
- Включает TRIM для SSD
- Отключает гибернацию
- Возвращает старый просмотрщик фотографий (по желанию)
- Включает классическое меню батареи (для ноутбуков, по желанию)
- Настраивает Game Mode для игр

## Как использовать скрипт

1. Сохраните exe м сmd (онм должны быть рядом в одной директории)
2. Запустите сmd
4. Подтвердите выполнение, введя "Да"
5. После завершения работы скрипта перезагрузите компьютер

## Важные предупреждения

- Перед запуском скрипта рекомендуется создать точку восстановления системы
- Некоторые изменения могут повлиять на работу определенных функций Windows
- Скрипт предназначен для опытных пользователей
- Автор не несет ответственности за возможные проблемы после выполнения скрипта

Скрипт подходит для Windows 10 и 11. Некоторые функции могут не работать в более старых версиях системы.
